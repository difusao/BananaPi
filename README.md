# INSTALL IMAGE RASPBIAN IN TO SD CARD

> On Linux
```
fdisk -l
dd if=/media/images/Raspbian_For_BananaPro_v1412.img of=/dev/sdb bs=1M && syn

```
> On Windows 10
Download https://sourceforge.net/projects/win32diskimager/
